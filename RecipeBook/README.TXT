HOW TO START THIS PROJECT:Recipe Manager Application
How to Run This Project
Prerequisites
You must have the Java Development Kit (JDK) installed (version 8 or higher).

A terminal (Command Prompt, PowerShell, or Mac Terminal) or a Java IDE (VS Code, IntelliJ, Eclipse).

Option 1: Running via Command Line (Recommended)
Follow these steps to run the application manually from your terminal:

Navigate to the folder: Open your terminal/command prompt and navigate to the directory where your .java files are saved.



cd path/to/your/project/folder
Compile the code: Run the following command to compile all the class files at once. If successful, no errors will appear.


javac *.java
Run the application: Start the program using the Driver class (which contains the main method).


java Driver
Option 2: Running via an IDE (IntelliJ, Eclipse, VS Code)
Open your IDE.

Import the project folder or copy all the .java files (Driver, Recipe, RecipeBook, BakedDish, etc.) into the src folder of a new project.

Open Driver.java.

Click the green Run / Play button.

USER INSTRUCTIONS:How to Use the Application
Once the application is running, you will see a text-based menu.

Seeded Data: The program starts with 2 recipes pre-loaded (a Chocolate Cake and a Greek Salad) so you can test the "List" and "Search" functions immediately.

Adding a Recipe: Select Option 1. You will be asked if the dish is Baked or Cold.

Baked Dishes ask for Oven Temp and Bake Time.

Cold Dishes ask for Chill Time.

Preparing: Select Option 5 to simulate "Cooking" the meal. This demonstrates the specific preparation steps based on the recipe type.

Scaling: Select Option 7 to change the serving size. This automatically recalculates the quantity of ingredients needed.
